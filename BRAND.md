# Azuki — 브랜드 디자인 시스템

마스코트 **아즈키(Azuki)** 의 시각 정체성을 정의한다. 여러 앱에서 공용 마스코트로 쓰이므로,
이 문서가 아즈키가 등장하는 모든 UI의 단일 기준이다.

> 이 문서는 마스코트 원화(`icons/azuki-art.png`)에서 도출했다. 확장이 **생성**하는 `DESIGN.md`(분석 대상 사이트용)와는 별개다 — 이건 우리 제품 자체의 브랜드다.

## 1. 무드 & 아틀리에

흑백 만화(少女漫画) 잉크 드로잉 + 단 하나의 시그니처 컬러(일렉트릭 블루). 차갑고 절제된 표정,
고대비 흑백 위에 파랑이 유일한 채색으로 얹혀 시선을 끈다. 키워드: **잉크, 절제, 고대비, 미니멀, 시크**.

- 색은 잉크 블랙과 크림 페이퍼가 지배하고, 블루는 **액션·강조에만** 아껴 쓴다.
- 장식·그라디언트·다중 그림자 없음. 종이 위 잉크처럼 플랫하게.
- 모서리는 거의 각지게 — 부드러움보다 또렷함.

## 2. 컬러 팔레트

| 토큰 | HEX | 역할 |
|------|-----|------|
| `--azuki-ink` | `#14161c` | 잉크 블랙. 본문 텍스트, 라인아트, 강한 요소 |
| `--azuki-ink-mute` | `#5b606e` | 보조 텍스트, 캡션, 헬퍼 |
| `--azuki-cream` | `#f6f3ec` | 크림 페이퍼. 기본 배경 |
| `--azuki-paper` | `#ffffff` | 카드·입력창 등 순백 표면 |
| `--azuki-hairline` | `#e3ddd0` | 1px 구분선·테두리 |
| `--azuki-blue` | `#2f6bff` | **Primary.** 시그니처 일렉트릭 블루. CTA·링크·강조 |
| `--azuki-blue-deep` | `#1e4fd0` | Primary hover/press |
| `--azuki-blue-soft` | `#6fa8ff` | 밝은 블루(눈동자). 포커스 링·미묘한 강조 |
| `--azuki-on-blue` | `#ffffff` | 블루 표면 위 텍스트 |

블루는 화면당 한두 곳(주요 액션·활성 상태)에만. 남용하면 잉크 대비가 죽는다.

## 3. 타이포그래피

- 시스템 산세리프 스택(`-apple-system, "Apple SD Gothic Neo", sans-serif`). 별도 웹폰트 없음 — 확장 팝업은 가볍게.
- 위계: 제목 15px/700 · 본문 13px/400 · 캡션 12px/400. 굵기는 400과 700만.
- 잉크 컬러로 또렷하게. 회색 본문 금지(무게감이 브랜드).

## 4. 형태 & 깊이

- **모서리 반경**: 버튼·입력 `6px`, 카드·패널 `10px`. pill 없음.
- **그림자**: 기본 없음. 필요 시 단 하나 `0 1px 0 rgba(20,22,28,.06)` 수준의 얇은 잉크 라인만.
- 구분은 그림자 대신 `--azuki-hairline` 테두리로.

## 5. 컴포넌트 규칙

- **Primary 버튼**: 배경 `--azuki-blue`, 텍스트 흰색, 반경 6px, hover 시 `--azuki-blue-deep`.
- **Secondary 버튼**: 배경 `--azuki-paper`, 잉크 텍스트, 1px `--azuki-hairline` 테두리, hover 시 크림 채움.
- **입력·코드 영역**: 잉크 배경(`--azuki-ink`)에 크림 텍스트로 만화 잉크 패널 느낌. 또는 페이퍼 배경.
- **포커스**: `--azuki-blue-soft` 2px 아웃라인.

## 6. Do / Don't

- **Do**: 크림 배경 + 잉크 텍스트 기본. 블루는 액션에만. 각진 모서리, 플랫 표면.
- **Don't**: 블루를 넓은 배경에 깔지 않기. 그라디언트·컬러풀한 팔레트 추가 금지. 둥근 pill·과한 그림자 금지.
