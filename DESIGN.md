# Azuki — 디자인 시스템

마스코트 **아즈키(Azuki)** 의 시각 정체성을 정의한다. 여러 앱에서 공용 마스코트로 쓰이므로,
이 문서가 아즈키가 등장하는 모든 UI의 단일 기준이다.

> ⚠️ 이 문서는 **우리 제품 자체**의 디자인 시스템이다. 확장이 분석 대상 사이트로부터
> **생성**하는 산출물 `DESIGN.md`와는 완전히 별개다. 혼동 금지.
> (원화 출처: `icons/azuki-art.png`)

## 1. 무드 & 아틀리에

흑백 만화(少女漫画) 잉크 드로잉 + 단 하나의 시그니처 컬러(일렉트릭 블루). 차갑고 절제된 표정,
고대비 흑백 위에 파랑이 유일한 채색으로 얹혀 시선을 끈다. 키워드: **잉크, 절제, 고대비, 미니멀, 시크**.

- 색은 잉크 블랙과 크림 페이퍼가 지배하고, 블루는 **액션·강조에만** 아껴 쓴다.
- 그라디언트·블러 그림자·다중 그림자 없음. 종이 위 잉크처럼 플랫하게.
- 모서리는 거의 각지게 — 부드러움보다 또렷함.

## 2. 시그니처 요소 (이게 Azuki다)

일반적인 "깔끔한 웹 UI"와 Azuki를 구별하는 요소. 새 화면·컴포넌트는 이 다섯 가지 위에서 시작한다.

| 요소 | 정의 | 토큰/값 |
|------|------|---------|
| **잉크 프레임** | 주요 패널·버튼은 만화 컷처럼 잉크 테두리로 감싼다. 연회색 1px 테두리는 보조 요소 전용 | `--azuki-frame: 1.5px solid var(--azuki-ink)` |
| **오프셋 잉크 섀도** | 블러 0의 단단한 오프셋 그림자 — 인쇄가 살짝 어긋난 판화 느낌. 부드러운 box-shadow 금지 | `--shadow-ink: 3px 3px 0 var(--azuki-ink)` / `--shadow-ink-sm: 2px 2px 0` |
| **프레스 모션** | 잉크 섀도 요소를 누르면 그림자 방향으로 밀려 들어간다(도장 찍기). hover는 색만, active는 위치 | `:active { transform: translate(2px,2px); box-shadow: none; }` |
| **스크린톤** | 크림 배경 위 만화 망점(halftone) 텍스처. 배경·칩에 아주 옅게 | `--screentone` (radial-gradient 도트, 8px 격자) |
| **말풍선** | 아즈키의 촌평은 잉크 프레임 말풍선 + 삼각 꼬리로. 마스코트가 "말한다"는 것을 형태로 보여준다 | `.bubble` 패턴 (popup.css `#mascot`) |

## 3. 컬러 팔레트

| 토큰 | HEX | 역할 |
|------|-----|------|
| `--azuki-ink` | `#14161c` | 잉크 블랙. 본문 텍스트, 프레임, 잉크 섀도 |
| `--azuki-ink-mute` | `#5b606e` | 보조 텍스트, 캡션, 헬퍼 |
| `--azuki-cream` | `#f6f3ec` | 크림 페이퍼. 기본 배경 |
| `--azuki-paper` | `#ffffff` | 카드·입력창 등 순백 표면 |
| `--azuki-hairline` | `#e3ddd0` | 보조 요소의 1px 구분선 (프레임과 위계 구분) |
| `--azuki-blue` | `#2f6bff` | **Primary.** 시그니처 일렉트릭 블루. CTA·링크·강조 |
| `--azuki-blue-deep` | `#1e4fd0` | Primary hover/press |
| `--azuki-blue-soft` | `#6fa8ff` | 밝은 블루(눈동자). 포커스 링·미묘한 강조 |
| `--azuki-on-blue` | `#ffffff` | 블루 표면 위 텍스트 |

블루는 화면당 한두 곳(주요 액션·활성 상태)에만. 남용하면 잉크 대비가 죽는다.

## 4. 타이포그래피

- 시스템 산세리프 스택(`-apple-system, "Apple SD Gothic Neo", sans-serif`). 별도 웹폰트 없음 — 확장 팝업은 가볍게.
- 위계: 제목 15px/700 · 본문 13px/400 · 캡션 12px/400. 굵기는 400과 700만.
- 기술 값(지문·코드)은 모노스페이스(`"SF Mono", Menlo, monospace`) + letter-spacing.
- 잉크 컬러로 또렷하게. 회색 본문 금지(무게감이 브랜드).

## 5. 형태 & 깊이

- **모서리 반경**: 버튼·입력 `4px`, 카드·패널 `8px`. pill 없음. 각짐이 기본값.
- **그림자**: 오프셋 잉크 섀도만(§2). 블러 있는 그림자·엷은 회색 그림자 금지.
- **깊이 위계**: 잉크 프레임+섀도(주요) > 잉크 프레임(중요) > hairline(보조). 세 단계뿐.

## 6. 컴포넌트 규칙

- **Primary 버튼**: 배경 `--azuki-blue`, 흰 텍스트, 잉크 프레임 + `--shadow-ink`, hover `--azuki-blue-deep`, active 프레스 모션.
- **Secondary 버튼**: 배경 `--azuki-paper`, 잉크 텍스트, 잉크 프레임 + `--shadow-ink-sm`, hover 크림 채움, active 프레스 모션.
- **아이콘/고스트 버튼**: 프레임·섀도 없음. hover 시 paper 채움만.
- **입력·코드 영역**: 잉크 배경(`--azuki-ink`)에 크림 텍스트로 만화 잉크 패널. 또는 페이퍼 배경 + 잉크 프레임.
- **칩(DNA 태그 등)**: 스크린톤 배경 + 잉크 프레임, 반경 2px.
- **말풍선(마스코트)**: 페이퍼 배경 + 잉크 프레임 + 좌상단 삼각 꼬리. 아즈키 촌평 전용.
- **포커스**: `--azuki-blue-soft` 2px 아웃라인.

## 7. Do / Don't

- **Do**: 크림+스크린톤 배경, 잉크 프레임, 오프셋 잉크 섀도, 프레스 모션. 블루는 액션에만.
- **Don't**: 블러 그림자·그라디언트·pill·컬러풀 팔레트 금지. 블루를 넓은 배경에 깔지 않기.
  잉크 프레임 없이 섀도만 쓰지 않기(프레임과 섀도는 한 세트).
